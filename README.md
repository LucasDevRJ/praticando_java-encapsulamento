# ☕ Praticando Java: Encapsulamento

Projeto desenvolvido durante o curso **Praticando Java: Encapsulamento**, da **Alura**, com o objetivo de aplicar os principais conceitos de **Programação Orientada a Objetos (POO)** utilizando a linguagem Java.

O repositório reúne diversos desafios práticos que simulam situações do cotidiano, colocando em prática o uso de encapsulamento, validações de dados, regras de negócio e organização do código.

---

## 📚 Objetivos do projeto

Durante o desenvolvimento deste projeto foram praticados conceitos fundamentais da Programação Orientada a Objetos, como:

* Encapsulamento
* Classes e Objetos
* Atributos privados
* Métodos públicos
* Getters e Setters
* Validação de informações
* Regras de negócio
* Utilização de `ArrayList`
* Uso de `enum`
* Organização em pacotes
* Boas práticas de estruturação de código

---

## 📂 Estrutura do projeto

Cada exercício foi desenvolvido em seu próprio pacote, simulando pequenos sistemas independentes.

| Pacote                        | Descrição                                             |
| ----------------------------- | ----------------------------------------------------- |
| Agenda de Contatos            | Cadastro e listagem de contatos.                      |
| Avaliação de Filmes           | Registro de avaliações e cálculo de média.            |
| Cadastro de Produto           | Validação de preço durante o cadastro.                |
| Controle de Saldo Bancário    | Depósitos, saques e consulta de saldo.                |
| Controle de Senha             | Alteração de senha mediante validação da senha atual. |
| Controle do Nível de Bateria  | Classificação do nível da bateria utilizando `enum`.  |
| Cadastro de Veículos          | Exercício de encapsulamento aplicado a veículos.      |
| Cadastro de Funcionário       | Manipulação de dados de funcionários e reajustes.     |
| Registro de Notas             | Armazenamento e cálculo da média de notas.            |
| Sistema de Pontos para Alunos | Evolução de nível conforme a quantidade de pontos.    |
| Validação de Login            | Simulação de autenticação com limite de tentativas.   |

## 🎯 Conceitos praticados

Durante a implementação dos exercícios foram utilizados recursos importantes da linguagem Java, como:

* Modificadores de acesso (`private` e `public`);
* Encapsulamento de atributos;
* Métodos de acesso (Getters e Setters);
* Validação de parâmetros antes da alteração do estado dos objetos;
* Estruturas condicionais;
* Estruturas de repetição;
* Coleções (`ArrayList`);
* Enumerações (`enum`);
* Organização do projeto em múltiplos pacotes.

## 🚀 Como executar

1. Clone este repositório.

```bash
git clone https://github.com/LucasDevRJ/praticando_java-encapsulamento.git
```

2. Abra o projeto em sua IDE Java de preferência.

3. Execute a classe `Principal.java` correspondente ao exercício que deseja testar.

---

## 📖 Aprendizados

Este projeto foi importante para consolidar um dos pilares da Programação Orientada a Objetos: **o encapsulamento**.

Ao longo dos exercícios foram implementadas validações diretamente nas classes, protegendo os atributos e garantindo que as regras de negócio fossem respeitadas antes da alteração dos dados dos objetos.

Além disso, o projeto proporcionou prática na organização de código em pacotes, reutilização de métodos, utilização de listas, enums e construção de pequenas aplicações orientadas a objetos.
